=====================================
Script Name:	The T0 Script Creator
=====================================

Written by:	Jason Murphy
Date:		12/30/2011
Revision:	1A

Purpose:	Will create the CREATES and DELETES for GR-303 T0s
		for DSL and PON subscribers

Specific Steps:	
		Logs into the C7 by IP address or hostname
		Retrieves all of the GR-303 T0 cross-connects on an IG indicated user-specified
		Then it creates a TXT file of the T0 creates
		Then it creates a TXT file of the T0 deletes
